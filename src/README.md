# Source Code for Rice ML

This folder contains the source code for `rice_ml`, which is further divided into supervised and unsupervised machine learning algorithms, as well as preprocessing and postprocessing of data.

### Contents
For further description, see the READMEs in the following packages from `rice_ml`:
- [`postprocess`](./rice_ml/postprocess/README.md): postprocessing modules
- [`preprocess`](./rice_ml/preprocess/README.md): preprocessing modules
- [`supervised_learning`](./rice_ml/supervised_learning/README.md): supervised learning algorithms
- [`unsupervised_learning`](./rice_ml/unsupervised_learning/README.md): unsupervised learning algorithms
